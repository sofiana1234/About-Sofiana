Why do I like anime?

I like anime because each one has a different genre which makes each one unique.
That means if you don't like one anime then you can try out another genre of anime!
I also like it because it helps improve my japanese. I watch anime in japanese so it helps me understand new words.
Another reason is that I can study how they animate.
I have been interested in animating so seeing different anime's and their different animating styles is very interesting for me.
